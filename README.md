# Netflix Content Analytics using Python

## Project Overview
This project analyzes Netflix movies and TV shows using Python to uncover insights about content distribution, genres, countries, and content growth trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Insights
- Netflix has significantly more Movies than TV Shows.
- The United States contributes the highest number of titles.
- International Movies and Dramas are among the most common genres.
- Netflix experienced rapid content growth after 2015.
